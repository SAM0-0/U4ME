# U4Me

[![GitHub Repo stars](https://img.shields.io/github/stars/SAM0-0/U4ME?style=social)](https://github.com/SAM0-0/U4ME)

This tiny app helps you see your SOH (State of Health) via Mahindra's me4u app infrastructure. 

*Note: We don't know how long it will last, but while it does, it's a great tool!*

### 100% Open Source & Trustworthy 🛡️
This project is completely open source! You don't have to blindly trust an APK file. You can check the source code yourself, verify there is no malicious code, and compile it directly from the source if you prefer. 

## Download
You can download the latest officially signed Release APK directly from the [Releases](https://github.com/SAM0-0/U4ME/releases) page.
